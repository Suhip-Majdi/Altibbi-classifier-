# Altibbi-classifier-
Logistic Regression, Support Vector Machines (SVMs), Decision Trees, Neural Networks

A multi-class disease classification system using text embeddings from TF-IDF, Word2Vec, and the NTFloat/multilingual-e5-large model from Hugging Face. This project enables accurate identification of disease names from medical text by leveraging both traditional and state-of-the-art semantic representations, supporting multilingual input and robust classification performance.
